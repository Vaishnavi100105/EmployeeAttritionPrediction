# Employee Attrition Prediction

## Project Overview
This project predicts whether an employee is likely to leave the company using Machine Learning.

## Dataset
IBM HR Analytics Employee Attrition Dataset (Kaggle)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Models Used
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Results
The models were evaluated using Accuracy, Precision, Recall, F1 Score, ROC-AUC, and Confusion Matrix. Feature importance analysis identified the key factors influencing employee attrition.
